# Transfer 穿梭框
在一定展示空间里对选项进行单个或批量移动从而完成挑选的数据容器。

### 何时使用
一组数据进行两种状态的分类时。
有更多的空间进行选择时。

## 1.组件类型
### 1.1 基础穿梭框

使用场景：需要两框之间的元素迁移时
