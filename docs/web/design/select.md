### 相似组件

| 组件名   | 何时使用                                                     |
| :------- | :----------------------------------------------------------- |
| 选择器   | 选择器主要用于收纳选项，是一种录入信息类控件，常用于表单、筛选等信息录入场景。 |
| 下拉菜单 | 下拉菜单主要用于对过多的操作进行收纳，常跟随于按钮后面，点击后是触发相应的操作。 |
