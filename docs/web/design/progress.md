### 相似组件

| 组件名 | 何时使用                                                               |
| :----- | :--------------------------------------------------------------------- |
| 进度条 | 页面局部或全局处于等待数据加载时，使用加载中让用户清晰感知到当前状态。 |
| 滑 块  | 在数值输入过程中，需要提供实时的视觉比例反馈时。                       |
