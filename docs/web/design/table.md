


### 何时使用

使用表格组织、展示大批量数据，易于用户浏览和获取，特别是展示大量信息内容的时候；

对数据进行排序、搜索、筛选、分页等操作时可以使用表格承载；

使用表格对数据进行归纳、分类，便于用户快速了解其中的差异与变化、关联和区别等。
