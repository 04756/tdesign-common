### 3.3.相似组件

| 组件名     | 何时使用                                                   |
| :-----     | :--------------------------------------------------------- |
| 级联选择器 | 一组数据进行两种状态的分类时；总类和子类的选项筛选时。     |
| 穿梭框     | 一组数据进行两种状态的分类时；总类和子类的选项筛选时。     |
| 树         | 用于承载有父子关系的结构化内容，提供内容层级的展示。       |
