### 3.3. 相似组件

| 相似组件             | 何时使用                                                     |
| :------------------- | :----------------------------------------------------------- |
| 警告 Alert | 用于承载需要用户注意的信息。 |
| 消息通知Notification | 需要提醒用户来自系统的消息，且不打断用户；带有解释描述的提醒内容；需要用户进行相关交互时，如：确认操作等。 |
| 全局提示Message      | 在完结某个独立页面后的反馈，如：付款成功页面；在一个操作区域或一系列操作完成之后的总体反馈，如：提交分步骤表单中的某个表单；在某个操作点之后的反馈，如：针对信息复制操作的结果反馈。 |
