### 相似组件

| 相似组件             | 何时使用                                                     |
| :------------------- | :----------------------------------------------------------- |
| [全局提示](./message)| 对用户的操作作出轻量的全局反馈。 |
| [警告](./Alert)  | 警告常用于承载站内相关产品某项功能的解释说明、使用注意事项等，置于内容展示区域顶部，非悬浮层。 |
| [消息通知](./Notification) | 需要提醒用户来自系统的消息，且不打断用户时；带有解释描述的提醒内容时。 |
