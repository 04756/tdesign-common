

## 1.组件设计指南

### 1.1. 使用规范

面包屑是一种辅助导航，出现的位置一般是页面的左上部分，而且主要是在导航下面或者内容区的上面。在页面中不可独立出现，如果出现以下情况，可以选择使用面包屑：

#### 1.1.1. 当系统超过两级以上层级结构；

#### 1.1.2. 当系统需要让用户了解自己所处位置时；

#### 1.1.3. 当需要有返回上一层级导航功能时。

![使用规范](https://tdesign.gtimg.com/site/design/images/使用规范-1818933.jpg)
