# 组件概览

TDesign 基于海量业务沉淀，为Web、Mobile、小程序等平台提供丰富组件库，持续赋能产业互联网，除提供官方基础组件外，TDesign 也提供各垂直领域的行业组件，满足全场景的生态能力建设。
