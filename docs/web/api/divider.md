# Divider 分割线

分割线是一个呈线状的轻量化组件，起到分割、组织、细化的作用，用于有逻辑的组织元素内容和页面结构。

### 何时使用

需要对不同章节的文本段落进行分割时；

需要对行内文字、链接等进行分割时。

## 组件类型

在TDesign中，提供2种不同形式的分割线：基础分割线和带文字的分割线。

### 基础分割线

基础分割线是没有文字的独立线条，又分为水平分割线和垂直分割线。
