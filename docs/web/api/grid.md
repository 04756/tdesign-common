# Grid 栅格

栅格是以规则的网格阵列来指导和规范网页中的版面布局以及信息分布，提高界面内布局的一致性，节约成本。

### 何时使用

栅格用于页面区域划分，稳定地呈现相对应的内容。

## 组件类型

为了平衡灵活性与复杂度，对于内容区域，我们采用12栅格系统。
