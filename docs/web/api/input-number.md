# InputNumber 数字输入框

数字输入框由增加\&减少按钮、数值输入组成。每次点击增加按钮（或减少按钮），数字增长（或减少）的量是恒定的。

### 何时使用

适用于连续数值的小范围精确调整，如商品数量、人数等的小范围调整，数字输入框具有优异表现。

连续数值是指一个等差数列时：如等差为10的数列：0，10，20，30，40，50，60，70.....
