# Tooltip 文字提示

用于文字提示的气泡框。

### 何时使用

鼠标移入需要展示文字解释说明时。

## 组件类型
在TDesign中，提供 5 种不同形式的按钮：带箭头的文字提示、不带箭头的文字提示、带主题色的文字提示、预支持的文字提示、触发状态的文字提示

### 带箭头的文字提示

带箭头的文字提示有较明确的指向性。常用于有多个需要提示的信息并列放置时，对某个具体信息进行提示。

{{ arrow }}

### 不带箭头的文字提示

不带箭头的文字提示没有明确指向性。常用于不需要针对性提示的场景中。
