# Loading 加载

定义：显示和提示页面和区块的加载中状态，出现时有动效。

### 何时使用

页面全局处于等待数据加载中，合适的加载动效有助于提升用户体验；

- 页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有助于缓解用户的焦虑。

## 1. 组件类型

### 1.1. 仅图标加载

定义：加载过程中只有图标显示。

使用场景：该组件为基础组件，适用于打开页面、操作完成后等待页面的全局/局部加载的场景。
