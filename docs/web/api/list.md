# List 列表

列表用一个连续的列来显示多行元素，常用于具有相同构成及内容的模块批量展示，可承载多样化的信息内容，从纯文字到复杂的图文组合。

### 何时使用

需批量展示具有相同构成及内容的模块时；

需展示多样的结构化的信息时，如同时展示图片、文字、操作等信息。

## 组件类型

### 基础文字列表

定义：仅包含简单文字的列表。

使用场景：对较简单的信息进行陈列时，如操作列表等。

### 多行文字列表

定义：仅包含主要文字及描述性文字的列表。

使用场景：对较复杂的，包含多个字段或内容的信息进行展示时。

### 基础图文列表

定义：包含简单图文的列表。

使用场景：需使用图片和文字结合展示信息。

### 带操作列表

定义：包含图文信息及操作的列表。

使用场景：需展示较复杂的信息结构，表格无法满足时，同时需要对所在列进行操作。
